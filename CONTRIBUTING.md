# 贡献指南

感谢你考虑为数据可视化展示平台做出贡献！🎉

## 🤝 如何贡献

### 报告问题
- 使用 [Issues](https://github.com/yourusername/data-visualization/issues) 报告bug
- 提供详细的重现步骤
- 包含截图和错误信息

### 提出新功能
- 在Issues中描述新功能需求
- 说明功能的使用场景和价值
- 讨论实现方案

### 提交代码
1. Fork项目
2. 创建功能分支：`git checkout -b feature/amazing-feature`
3. 提交更改：`git commit -m 'Add some amazing feature'`
4. 推送分支：`git push origin feature/amazing-feature`
5. 创建Pull Request

## 📋 开发规范

### 代码风格
- JavaScript使用ES6+语法
- HTML使用语义化标签
- CSS遵循BEM命名规范
- 添加适当的注释

### 提交信息格式
```
type(scope): description

[optional body]

[optional footer]
```

类型说明：
- `feat`: 新功能
- `fix`: 修复bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建过程或辅助工具的变动

示例：
```
feat(chart): add zoom functionality to wind speed chart

- Implement pinch-to-zoom on mobile
- Add zoom controls for desktop
- Update chart tooltips during zoom

Closes #123
```
